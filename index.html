Jasne, oto przykładowy kod CSS dla strony internetowej. Możesz go dostosować do swoich potrzeb: ```css /* Resetowanie stylów */ * { margin: 0; padding: 0; box-sizing: border-box; } /* Stylizacja ciała strony */ body { font-family: Arial, sans-serif; line-height: 1.6; background-color: #f4f4f4; color: #333; } /* Nagłówek */ header { background-color: #333; color: #fff; padding: 1rem 0; text-align: center; } /* Sekcja główna */ main { padding: 20px; } /* Artykuły */ article { background: #fff; padding: 15px; margin-bottom: 20px; border-radius: 5px; box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1); } /* Tytuły artykułów */ h2 { margin-bottom: 10px; color: #333; } /* Paragrafy */ p { margin-bottom: 15px; } /* Stopka */ footer { text-align: center; padding: 1rem 0; background-color: #333; color: #fff; font-size: 0.8rem; } /* Klasy pomocnicze */ .container { width: 80%; margin: auto; overflow: hidden; } .btn { display: inline-block; background: #333; color: #fff; padding: 10px 20px; text-decoration: none; border-radius: 5px; } /* Responsywność (przykład) */ @media (max-width: 768px) { .container { width: 90%; } } ```
